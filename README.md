## 설명
사용자의 선택에 따라 활동을 추천해주는 콘솔 기반 CLI 프로그램입니다.
외출 여부와 세부 활동을 단계적으로 선택하면,
최종적으로 해당 활동이 추천되고 간단한 메시지가 출력됩니다.

### 프로그램 흐름
1. 사용자에게 외출 / 집콕 여부를 묻는다.
2. 선택에 따라 세부 활동 분기로 이동한다.
3. 최종 선택된 활동 객체를 생성한다.
4. recommend() 메서드를 호출하여 활동 추천 메시지를 출력한다.
